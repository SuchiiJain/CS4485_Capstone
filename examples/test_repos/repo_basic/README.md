# Basic Test Repo

## Auth

- `login(username, password)` — Authenticates user credentials.
- `logout(user_id)` — Clears user session.
- `get_user(user_id)` — Returns user record by ID.

## Utils

- `format_name(first, last)` — Returns formatted full name.
- `clamp(value, min_val, max_val)` — Clamps an integer within a range.
